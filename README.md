Sistem za urejanje naročil

Opis projekta
Program sistem za urejanje naročil je namizna aplikacija, razvita v programskem jeziku C#. Namen programa je omogočiti enostavno upravljanje naročil tekstilnih izdelkov, kot so kratke majice in puloverji. Uporabnik lahko vnese svoje osebne podatke, izbere izdelek ter določi njegove lastnosti. Program samodejno izračuna ceno izdelka in omogoča prikaz celotnega naročila.

Funkcionalnosti
- Vnos podatkov o kupcu
- Shranjevanje aktivnega kupca
- Izbira izdelka
- Izbira barve in velikosti izdelka
- Izbira tiska pri majici
- Izbira zadrge pri puloverju
- Samodejni izračun cene izdelka
- Dodajanje izdelkov v naročilo
- Prikaz vseh izdelkov v naročilu
- Izračun in prikaz skupne cene naročila

Namestitev
1. Odprite repozitorij na portalu GitHub.
2. Kliknite na gumb code.
3. Izberite možnost download ZIP.
4. Preneseno datoteko razširite.
5. Odprite mapo projekta.

Zagon programa
1. Odprite datoteko s končnico .sln v programu Microsoft Visual Studio.
2. Počakajte da se projekt naloži.
3. Program zaženite s tipkami Ctrl + F5 ali s klikom na gumb start.

Uporaba programa
1. Uporabnik vnese svoje podatke.
2. Če so podatki pravilno vneseni se kupec shrani kot aktivni kupec.
3. Uporabnik izbere izdelek.
4. Nastavi lastnosti izdelka.
5. Izdelek doda v naročilo.
6. V seznamu naročila se prikažejo vsi dodani izdelki.
